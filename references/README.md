# References

This folder stores the project's tracked reference PDFs and research memos.

Current contents:

- `2014商事制度改革_政策背景与识别合理性.pdf`: policy background and identification memo for the 2014 Business Registration Reform design.
- `外部产品市场冲击与劳动市场势力_研究项目说明.pdf`: method reference for exposure construction. The current project does not use the 2018 tariff-war story as its economic narrative.
- `PDBCLS.pdf`: Gouin-Bonenfant (2022) reference on productivity dispersion, between-firm competition, and labor share.
- `AERI_2024_0570_final (3).pdf`: Rubens-Wu-Xu reference on wage markdown identification and labor-augmenting productivity.
